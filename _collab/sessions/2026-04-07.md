# NXT LINK — Session Notes: 2026-04-07

## What was done

Completely rewrote `src/app/page.tsx` from the old data-dashboard format into a new **Story Mode** design.

## Core change

The old page was a data dashboard — stat boxes, quick nav grid, and 3 decision cards. Users saw numbers but had no clear path to action.

The new page is a guided narrative. Users scroll through the story of what's happening in tech, who's driving it, and can contact companies or buy products at every step — like an Apple product page that sells.

---

## New Page Structure (8 sections, top to bottom)

### Section 1 — The Headline Story
- Full-width hero pulling `title`, `cause`, and `why_el_paso` from `/api/decide` top decision
- Live signal count badge (watches `total_signals_analyzed` from the API)
- "Explore the full story →" CTA button that smooth-scrolls to Section 3
- Loading state: skeleton blocks
- Right column: animated signal count badge + current urgency card

### Section 2 — Sector Pulse
- Title: "Where is tech moving right now?"
- 8 animated horizontal bars in a 2-column grid (mobile: 1 column)
- Sectors: AI/ML, Defense, Cybersecurity, Border Tech, Logistics, Manufacturing, Energy, Space
- Bar fill animates on scroll entrance via `useInView` + Framer Motion
- Static plausible percentages — rising = teal, falling = muted, neutral = dim
- Arrow indicators per sector

### Section 3 — The Big Story (3 narrative cards)
- Title: "What is happening"
- 3 cards from `/api/decide` top 3 decisions
- Each card: urgency badge + sector badge → title → cause → effect → divider → WHY EL PASO CARES (italic teal) → divider → WHAT TO DO → vendor pills → "Contact Vendors →" teal button linking to top vendor's website
- Loading: 3 skeleton cards (pulse animation)

### Section 4 — Where It's Heading
- Title: "What comes next"
- 3 static forward-looking cards (hardcoded compelling copy):
  1. AI + Defense convergence / Fort Bliss as AI test site
  2. US-Mexico border as technology corridor / $126B trade
  3. Quantum computing moving from lab to deployment
- Dark cards with gradient backgrounds, no buttons needed

### Section 5 — Companies Doing It
- Title: "The companies building this"
- 6 vendor cards in a 3-column grid (2 on tablet, 1 on mobile)
- Static data: Jacobs Solutions, Booz Allen Hamilton, Palantir, CrowdStrike, Shield AI, MesaAI
- Each card: `clearbit.com/{domain}` logo with colored initials fallback → company name → sector badge → IKER score (green) → 2-line description → "Visit Website →" + "Learn More" buttons
- "See all 442 companies →" link to `/vendors`

### Section 6 — Products You Can Buy
- Title: "Products available today"
- 4 product cards in a 2-column grid
- Products: Samsara Fleet Management, CrowdStrike Falcon, Palantir AIP, DAT Load Board
- Each: category badge + MATURE/EMERGING badge → product name → company → 3-line description → pricing → "Get Demo →" + "Compare" buttons
- "Browse all 1,041 products →" link to `/products`

### Section 7 — What to Watch (Conferences)
- Title: "What is coming up"
- 3 conference cards in a 3-column grid
- Static data: Hannover Messe 2026 (13 days), AUSA Annual 2026 (188 days), Border Security Expo 2026 (36 days)
- Each: countdown badge ("In X days" or "LIVE NOW") → conference name → location + dates → category badge → "Register →" (external) + "See Exhibitors" (internal) buttons

### Section 8 — Ask Anything (Search)
- Title: "Ask anything"
- All existing search functionality preserved exactly: `handleSearch`, `clearSearch`, `SearchResponse` display with cause/effect/consequence/actions/vendors/signals
- Same premium search bar with glow effect on focus

### Signal Ticker (bottom strip)
- Preserved exactly from the original — live marquee of recent signals from `/api/briefing`
